# DAYCARE MANAGEMENT SYSTEM

A desktop application for managing daycare operations (registration, records, and daily administration). Built with VB.NET / Visual Studio.

## Features
- Child registration and profile management
- Parent/guardian records
- Attendance tracking
- Fees / billing (as implemented)
- Reports (as implemented)

1. Open `DAYCARE_MANAGEMENT_SYSTEM.sln` in Visual Studio
2. Restore dependencies (if prompted)
3. Build and run the project
